# Raíces de Córdoba

**Raíces de Córdoba** es una API desarrollada con **NestJS** que busca impulsar el comercio local y preservar la identidad artesanal de la región de Córdoba, Colombia.  
A través de esta plataforma, los productores artesanales pueden registrar, gestionar y difundir sus productos, fomentando la economía regional y la conexión entre artesanos y consumidores.

---

## Tecnologías utilizadas

- **NestJS** – Framework para construir aplicaciones backend escalables en Node.js  
- **TypeScript** – Lenguaje tipado para mayor robustez y mantenibilidad  
- **PostgreSQL** – Base de datos relacional para almacenamiento seguro y eficiente  
- **PNPM** – Gestor de paquetes rápido y optimizado  
- **Swagger** – Documentación automática de la API
